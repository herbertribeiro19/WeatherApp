## WeatherApp - (IOS and Android)
The user can see the weather about some city in the world. The application will return the temperature, wind and humidity.

## Skills
- React Native
- TypeScript
- API - WeatherAPI
- Design by me
- IOS development App - MacOS
- Git & GitHub
